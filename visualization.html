1. The ggplot Recipe

The Concept: Making a graph in R is like making a cake. You have to add layers one by one using the + sign.

The 3 Layers:
1. Data: ggplot(data)
2. Aesthetics (aes): aes(x = col, y = col)
3. Geometry (geom): + geom_point()

Warning: Do NOT use the pipe |> here. Graphs use +.

2. Histogram (1 Numeric Var)

Shows the distribution of a single number column (e.g., "How heavy are the pumpkins?").

Syntax: ggplot(df, aes(x = numeric_col)) + geom_histogram()

Shows counts of categories (e.g., "How many students in each House?"). R counts the rows for you.

Syntax: ggplot(df, aes(x = category_col)) + geom_bar()

4. Boxplot (Num vs Cat)

Comparing a number across different groups (e.g., "Study Hours" by "School").

Syntax: ggplot(df, aes(x = category, y = number)) + geom_boxplot()

6. Color vs Fill

Taylor Doose is picky about colors. You need to know the difference:

  • color: Changes the outline (dots, lines).
  • fill: Changes the inside (bars, boxplots).
Syntax: aes(x = type, fill = type)

8. Faceting (Small Multiples)

Sometimes you have too much data for one chart. Faceting splits your chart into multiple panels, one for each group.

Syntax: + facet_wrap(~ grouping_variable)

Note: You MUST use the tilde ~ (read as "by").
